# Netflix Content Analytics Dashboard

## Overview
This Power BI dashboard analyzes Netflix movies and TV shows.

## Features
- Total Titles Analysis
- Genre Distribution
- Rating Analysis
- Country-wise Content Analysis
- Movies vs TV Shows Comparison
- Release Year Trends

## Tools Used
- Power BI
- Power Query
